what is the porpuse of get&post ?

Ans:1. GET Method Purpose: The GET method is used to request data from a specified resource. When you submit a form using GET, the form data is appended to the URL as query parameters. 2. POST Method Purpose: The POST method is used to submit data to be processed to a specified resource. Unlike GET, POST does not append data to the URL; instead, the data is sent in the request body.
